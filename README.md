# Alpha Snake

Static browser game ready for free Vercel deployment.

## Files

- `index.html`: the game
- `vercel.json`: static deployment config

## Deploy on Vercel

### Option 1: Vercel dashboard

1. Push `C:\test3` to a GitHub repository.
2. In Vercel, create a new project and import that repository.
3. Keep the root directory as the repo root.
4. Framework preset can stay as `Other`.
5. Deploy.

### Option 2: Vercel CLI

From this folder:

```powershell
npx vercel
```

For a production deployment:

```powershell
npx vercel --prod
```
